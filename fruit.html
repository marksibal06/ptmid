HTML Attributes The src Attribute
    There are two ways to specify the URL in the src attribute
  1. Absolute URL - Links to an external image that is hosted on another website.

    2. Example: src="https://www.w3school.com/img_girl.jpg"

    Notes: EXternal image might be under copyright, if you do not get permission to use it. you may be violation of copyright laws. In addition, you cannot control external images; it can suddenly be removed or changed

  2. Relative URL - links to an image that is hosted within the website. Here, The URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page.

    3. Tip: it is always best to use realative URLs. They will not break if you change the domain.
