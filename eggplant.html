HTML Attributes

The src Attributes

The <img> tag is use to embed an image in an HTML page.

The src attributes specifies the path to the image to be displayed.

• The <img> tag should also contain the width and Height attributes, which specifies of the image (in pixels).

• The required alt attributes for <image> tag specified an alternate text for and image, if image for some reason cannot be displayed. This can be due to slow connection, or an error in the src attribute, of if the user user screen reader
